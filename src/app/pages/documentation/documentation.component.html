Collabor IQ: Documentation
Get Started

Collabor IQ is an advanced chat application designed for Customer Service Agents, aiming to streamline and enhance user interaction by leveraging powerful AI capabilities. It acts as a sophisticated replacement for traditional customer service roles, equipped with various tools and knowledge bases to empower agents. Collabor IQ is primarily intended for customer service teams integrated into websites, delivered as an iframe for seamless consumption by existing web platforms.

Key Features
  • Main Dashboard (Overview): View agent and chatbot performance, resolution rate, agent status, average response time, error rate, question categories, top questions, and unanswered queries.
  • Real-Time Monitoring: Live conversation view, RAM/CPU/API metrics, and activity logs.
  • Prompt Editor & Agent Parameterization: Text-based prompt editing with templates and parameter controls.
  • Knowledge Base Management: Upload PDFs/CSVs, tag content, and visualize knowledge graphs.
  • User Feedback & Analytics: In-chat feedback, keyword analytics, error reports.
  • Multi-Tenant Support: Role-based access, client onboarding UI, and data separation.
  • Playground: Interactive testing with multi-agent chat and history view.
  • Admin Chat: Admins can chat directly with users.

Getting Started

To set up Collabor IQ in your local development environment:

npm install -g @angular/cli

git clone [Your_Git_Repository_URL]
cd Collabor-IQ
npm install
ng serve

The application will be available at http://localhost:4200/.
