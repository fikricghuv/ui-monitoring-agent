Get Started

TalkVera is a customer service chatbot platform powered by AI agents, designed to enhance user engagement, reduce manual workload, and deliver consistent and human-like interactions. The chatbot is provided via iframe for easy integration into client websites. It also includes admin tools for monitoring, editing, and analyzing conversations in real-time.

Key Features
  • Chatbot Embed via Iframe: Easily integrate the chatbot into your website using iframe. It works 24/7 to answer user queries instantly and intelligently.
  • Dashboard Analytics: Provides high-level metrics such as total interactions, active users, average response time, top intents, and bot-to-human resolution rate.
  • Prompt Editor: Customize the tone, language style, and behavior of the chatbot to match your company's brand voice.
  • Base-Knowledge Manager: Upload and manage reference documents (PDF, DOCX, etc.) that the chatbot will use to answer questions accurately and contextually.
  • Real-Time Monitoring: View ongoing conversations between users and the chatbot, complete with timestamps and sender information. Allows for direct intervention when needed.
  • Admin Chat: Human agents can jump into a chat session when the AI fails to handle the request. Enables seamless transition from bot to human support.
  • Feedback Analytics: Collects feedback and detects complaint signals to help you analyze user satisfaction and improve chatbot responses.
  • Customer Profile: Stores essential user data such as name, session history, and interaction patterns for follow-ups and personalized service.
  • Report Export: Download reports in CSV or PDF format, including conversation logs, usage metrics, and feedback summaries.
  • Playground (Internal Testing): A sandbox environment to test prompts, simulate interactions, and preview multi-agent responses before going live.
Technology Stack
  • Frontend: Angular v19, PrimeNG v19, Tailwind CSS, Google Fonts (Poppins).
  • Backend: FastAPI (Python).
  • Database: PostgreSQL (supports vector search with pgvector).
  • WebSocket: Real-time communication powered by WebSockets via FastAPI.
Project Structure
  • src/app/layout: Main layout components like sidebar and topbar.
  • src/app/pages: Application pages (Dashboard, Prompt Editor, Playground, etc.).
  • src/app/components: Shared components (Chat, Feedback, Documentation).
  • src/app/services: Application services (ChatCoreService, SessionService, etc.).
  • src/app/models: Shared data structures (e.g., User, MessageModel).
  • src/assets/demo: Assets for demo and testing.
  • src/assets/layout: SCSS themes and layout files.
  • src/assets/images: Avatars and image assets used in the interface.
