Get Started

Collabor IQ is an advanced chat platform designed to assist customer service teams in handling user interactions more efficiently. It leverages AI agents to answer customer questions, track service performance, and support real-time interventions. Delivered via iframe, Collabor IQ integrates seamlessly with websites to provide automated, responsive, and personalized customer service experiences.

Key Features
  • Chatbot Agent (Embed): A chatbot widget embedded on your website that interacts with visitors, answers questions, and operates 24/7.
  • Main Dashboard (Overview): Displays overall performance metrics such as total users, total chats, token usage, feedback counts, latency, and top frequently asked questions.
  • Real-Time Monitoring: Provides a live stream of all conversations, showing sender, message content, and timestamps to monitor active sessions as they happen.
  • Prompt Editor: Allows customization of chatbot behavior and response tone by editing prompt settings specific to your brand or communication style.
  • Knowledge Base Management: Upload reference documents (PDFs, etc.) to teach the chatbot. The agent uses this knowledge to respond more accurately to user inquiries.
  • Admin Chat: Enables admin users to directly enter live conversations, take over when necessary, or provide manual assistance to users in real time.
  • User Feedback & Analytics: Collects feedback submitted by users and visualizes issues, complaint keywords, timestamps, and performance patterns.
  • Users Profile: Stores user-submitted identity data such as name, email, and phone number, which can be used for session tracking or follow-up actions.
  • Report Export: Export various data including chat logs, token usage, and user feedback into formats such as CSV, Excel, or PDF for reporting and analysis.
  • Playground: A testing area for internal users to experiment with multi-agent chat flows and simulate real scenarios (optional for production use).
Technology Stack
  • Frontend: Angular v19, PrimeNG v19, Tailwind CSS, custom CSS, Google Fonts (Poppins).
  • Backend: FastAPI
  • Database: PostgreSQL
  • WebSocket: Real-time chat powered by WebSockets (FastAPI).
Project Structure
  • src/app/layout: Main layout files and components (sidebar, topbar, etc).
  • src/app/pages: Pages like Dashboard, Prompt Editor, Playground, etc.
  • src/app/components: Reusable components (Documentation, Chat, Playground).
  • src/app/services: Services like ChatCoreService and SessionService.
  • src/app/models: Shared data models (e.g., MessageModel).
  • src/assets/demo: Demo-specific assets.
  • src/assets/layout: SCSS layout and theme files.
  • src/assets/images: Avatars and image assets.
